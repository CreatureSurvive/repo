libCSColorpicker

  • A simplistic yet powerful color picker for use in PreferenceLoader preference bundles. Supports iOS 9 - 11, other iOS versions have not been tested.

    Best when used with libCSPreference but also works fine on it’s own. For usage info and resources, check out libCSColorPicker on GitHub, soon to be open sourced.

    v1.0 API changes

    libCSPreferences v1.0 has been released. in this release the API has been reworked to resolve conflicts with other projects, the legacy API has been officially marked depreciated and should not be used in new/updated projects. Legacy API will remain operational for now to ensure no projects are broken with this update. Please Download the new project resources for theos and update your projects.
